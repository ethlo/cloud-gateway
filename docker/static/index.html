Features

Built on top of Spring Cloud Gateway, giving it a powerful feature set:

  • Built on Spring Framework and Spring Boot.
  • Able to match routes on any request attribute.
  • Predicates and filters are specific to routes.
  • Circuit Breaker integration.
  • Spring Cloud DiscoveryClient integration.
  • Easy to write Predicates and Filters.
  • Request Rate Limiting.
  • Path Rewriting.
This project further adds additional features:
  • Full request and response logging, even when the upstream target service is down.
  • An in-memory rate-limiter implementation, avoiding setting up additional dependencies like Redis.

Generate traffic

The default example configuration is set up to allow proxying to https://httpbin.org

The sample configuration also requires that we present a username/password using basic-auth or using a JWT token, as the user identity is what is used to control the rate-limiting.

POST 
curl -v -H 'Content-Type: application/json' --data "{'key':'some data'}" http://localhost:6464/anything -ujohn:mypass
GET 
curl -v -H 'Content-Type: application/json' http://localhost:6464/anything -ujohn:mypass
Generate more traffic 
ab -A"john:pass" -n10 http://localhost:6464/anything
